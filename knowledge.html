🧠 AI知识卡片

把晦涩的AI技术概念，变成人人能懂的知识卡片。
正面是术语，背面是通俗解读——像翻扑克牌一样学习AI。

🤖
LLM
Large Language Model

大语言模型。像ChatGPT这样的AI，能理解和生成人类语言。 把它想象成一个读过亿万本书的"超级阅读者"，能回答问题、写文章、甚至写代码。

🎯
Fine-tuning
微调

就像给AI"上专业课"，在通用能力基础上，针对特定领域（如医疗、法律）进行额外训练，让它在该领域表现更专业。

🧠
Transformer
变换器架构

AI模型的核心"大脑结构"，通过"注意力机制"理解上下文关系。 想象读书时，你能同时关注这句话和前面的关键词，Transformer就是这样"理解"文本的。

👁️
Attention
注意力机制

让AI知道"重点看哪里"的机制。就像你读句子时会注意到关键词， Attention让模型自动找到文本中最重要的部分，决定信息如何流动。

⚡
RAG
检索增强生成

Retrieval-Augmented Generation。AI先去"查资料"，再回答问题。 像考试时允许翻书，答案更准确、更有依据。

📊
Quantization
量化

把模型"瘦身"。从32位精度降到8位甚至4位，让大模型能在普通电脑上运行。 像把高清视频压缩成小文件，画质略有损失但能流畅播放。

🔥
LoRA
低秩适配

一种高效的微调方法。不改动整个模型，只添加一小块"补丁"。 像给手机换壳而不是重装系统，节省99%的计算量。

💬
Prompt
提示词

给AI的"指令"。如何提问才能得到想要的答案？ Prompt工程就是设计精准指令的艺术，像写代码一样和AI对话。

🔄
Inference
推理

AI模型"运行"的过程。训练是学习，推理是应用。 像学生考试时运用学到的知识解题，模型推理就是生成答案的过程。

🌐
Embedding
嵌入向量

把文字变成"数字坐标"。让计算机理解词语之间的关系。 "猫"和"狗"在向量空间里距离很近，"猫"和"汽车"距离很远。

📝
Context
上下文窗口

AI能"记住"的对话长度。Context Window越大，能参考的信息越多。 像手机的RAM容量，决定一次对话能承载多少信息。
